Seed

Optional. Set the same seed (plus identical phonology settings) on two devices to have them independently generate the exact same Brunch words for the exact same English words — like a Minecraft world seed — without ever sharing the brunch.db file itself. Leave blank for fully random generation.

Changing the seed only affects words translated after saving. Clear the dictionary first if you want existing words to regenerate under the new seed.
